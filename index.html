NARWAY Bot

Narway bot is something I coded just for fun. One of my friend's birthdays was coming up at covid was still at its peak. Considering everything, I thought of making an online treasure hunt.

Narway bot(whose name is a comic reference to how one of our Math teachers misspronounced Norway) is coded in Ruby and hosted on BitBucket. It was never supposed to serve a large audience so I used my laptop to act as its server.

The bot greets those who say "Hello" and provides them with appropriate links for further treasures.

Oasis

We have designed a platform for students to interact, talk about their mental, social and physical health, and get consultancy if required. The website includes a portal to spread awareness about self-care, to have a discussion on common health issues, and a chatbot to connect individuals with doctors. The intuitive design helps the students navigate freely through the interface. There is a plan to strengthen the students' mental and social health by having weekly talk sessions. We also have an option for colleges to connect to our network.
The website also includes links to the already available tips by professionals. We have also included two mind-boggling games on our page.

I built the front-end of the website including the landing page, the menus and various sections, excluding the chatbot featured on the main page. The games are open-sourced codepens.

The landing page is made using ReactJS and is hosted on firebase. The rest of the pages use Vanilla framework and are hosted on GitHub pages. The code is available on GitHub.

Project Name

I built this project on Google Colab. I selected VGG-19 because of it versatile nature and the ability to identify and work with images. VGG-19 is a convolutional neural network that is 19 layers deep. One can load a pretrained version of the network trained on more than a million images from the ImageNet database. The pretrained network can classify images into 1000 object categories, such as keyboard, mouse, pencil, and many animals.

Music Generation using LSTM

I worked on this project with four of my classmates. We demonstrated how to generate musical notes using a simple recurrent neural network (RNN). We trained a model using a collection of piano MIDI files from the Alan Walker music(MIDI). We have used the Long Short Term Memory(LSTM) technique, i.e., given a sequence of notes, the model will learn to predict the next note in the sequence.

My role was to optimise the code and alter the numeric calculations to prevent repeated or blank notes in the output.
